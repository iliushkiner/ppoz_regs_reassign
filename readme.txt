Необходимо включить Режим разработчика в настройках расширений chrome://extensions/
После перетащить папку или загрузить распакованное расширение.
В настройках расширения есть возможность отключения всплывающего при наведение на номер обращения окна истории назначения обращения.
При клике на иконку загружается статистика по отделу для этого необходимо выбрать в фильтре регион и отдел.
