Список обращений назначаемых на рега

Группа: используется для группировки при выводе стаистики назначений.
Безлимит: указывается при условий назначений определенных опращений только указанным регам, без учета лимита и количества распределения.
Список регов: указывается в массиве через запяту в виде [{"login":"agtanaev","percent":100,"limit":40,"include":"on"},...]
  • login - учетная запись
  • percent - процент назначения от общего количества
  • limit - максимальное количество в день назначений
  • include - включение в распределение, где on - включить и off - исключить
JSON фильтр: свойства подходящийх обращений. Можно посмотреть в закладке Network инструмента DevTools браузера Chrome в ответе на запрос http://ppoz-service-bal-01.prod.egrn:9001/manager/requests/byId?id=PKPVDMFC-2021-01-20-614725
